## Codecount
A command line utility for counting and diplaying lines of code in a project folder by language

# Supported Languages
Rust,
Go,
C++,
C,
C/C++ Headers,
C#,
Dart,
Bash,
Python,
Julia,
Java,
JavaScript,
GDScript,
Kotlin,
Lua

# Usage

-> Add codecount to path
-> Open terminal
-> Change into project directory
-> Enter 'codecount' and click enter
